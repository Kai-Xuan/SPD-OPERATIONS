
Covariance Computation via four metrics on the Symmetric Positive Definite(SPD) manifold.  
Four metrics:  
              1.Affine Invariant Riemannian Metric(AIRM),  
              2.Stein divergence,  
              3.Jeffrey divergence,  
              4.Log-Euclidean Metric(LEM).
